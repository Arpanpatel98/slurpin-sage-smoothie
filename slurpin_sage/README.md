## 📁 Project Setup

Follow the steps below to run the project locally:

---

### 2. Navigate to the Project Folder

```bash
cd slurpin_sage
```
### 3. Install Dependencies
```bash
npm install

```
---
### 4. Run the Development Server
```bash
npm run dev

```
---
### 5. Open in Browser
```bash
http://localhost:5173/
```
